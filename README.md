# Поисковик файлов (FileSearcher)

Консольное приложение на C#/.NET для рекурсивного поиска файлов с расширениями `.txt` и `.cs` в указанном каталоге.

---

## Описание

Программа сканирует указанную папку и все её подпапки, находит все файлы с расширениями **.txt** и **.cs**, выводит информацию о них в удобной таблице и сохраняет результаты в CSV-файл.

### Возможности:
- Рекурсивный поиск по всему дереву каталогов
- Поиск только файлов `.txt` и `.cs`
- Вывод красивой таблицы в консоль (имя, размер, дата изменения)
- Сохранение результатов в `file_search_result.csv` (с разделителем `;`)
- Поддержка передачи пути через аргумент командной строки
- Обработка ошибок (пропускает недоступные папки)
- Автоматическое использование текущей папки, если путь не указан

---

## Технологический стек

1. C# — язык программирования
2. .NET 8 — платформа
3. `System.IO` — работа с файловой системой
4. `FileInfo` — получение информации о файлах (размер, дата)
5. `StreamWriter` + UTF-8 — запись в CSV
6. Рекурсивный обход директорий
