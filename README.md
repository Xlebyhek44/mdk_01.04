# Поисковик файлов (FileSearcher)

**МДК 01.04 — Системное программирование**

Консольное приложение на C#/.NET, которое рекурсивно ищет файлы с расширениями `.txt` и `.cs` в указанном каталоге и всех его подпапках.

---

## Описание

Программа выполняет следующие задачи:
- Рекурсивный обход всех подпапок
- Поиск только файлов с расширениями **.txt** и **.cs**
- Вывод информации в удобной таблице (имя файла, размер, дата изменения)
- Сохранение результатов в файл `file_search_result.csv`
- Игнорирование служебных папок `bin` и `obj`

---

## Технологии и возможности

- **Язык:** C#
- **Платформа:** .NET 8
- **Работа с файловой системой:** `System.IO`, `FileInfo`
- **Экспорт данных:** CSV (разделитель `;`, кодировка UTF-8)
- Поддержка запуска с аргументом командной строки
- Обработка ошибок доступа к папкам
